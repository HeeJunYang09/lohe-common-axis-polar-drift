# Generated Figures

Run the scripts from the repository root to generate the final manuscript
figures in this directory. The scripts produce PDF, PNG, and EPS formats.
The public repository retains the final PDF figure outputs.

The final manuscript PDF filenames are documented in
`paper/figure_mapping.md`.

Figure 6 is generated by `scripts/fig06_d5_block_selection.py` and saved as
`fig6_d5_block_selection.pdf` for the manuscript.
Panel (a) annotates the representative `K=10` case as satisfying the primary
`C_tol=5` criterion at `t=0`; positive crossing markers are shown only when a
positive interior fast time is detected.
Panel (c) compares the full and reduced drift coefficients; in the current
processed artifacts the drift relative L2 error is approximately `2.1e-4` to
`8.1e-4`. Panel (d) shows the reduced polar law only on the prescribed polar
validity window.

Appendix d5 figures generated by `scripts/appendix_d5_diagnostics.py` are
`appendix_figA1_d5_ansatz_validation` and
`appendix_figA2_d5_controls_robustness`.
